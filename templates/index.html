Wikirace

Open-source сингл-/мультиплейрная онлайн игра для соревновательных вики-гонок

Info

Что такое вики-гонки

Wikiracing — это игра, использующая онлайн-энциклопедию Википедия, в которой, используя ссылки на другие статьи, игроки переходят с одной страницы на другую. Узнать больше

Функциональности проекта

Этот веб-сайт является образовательным проектом на 1-м курсе ПМИ в Питерской Вышке. Больше информации на Github странице.

Асинхронные веб-сокеты

Для обмена данными во время игры используются асинхронные веб-сокеты. Это обеспечивает игровой процесс без задержек и не требует неэффективных и запутанных полл-запросов.

База данных

Этот проект использует Postgresql базу данных как основное хранилище данных, а также использует Redis для хранения кратко-срочной информации для веб-соокетов. Обе эти базы данных управляются Heroku и позволяют автоматический scaling.

Авто-деплой и CI/CD

Деплой and CI/CD обеспечено платформой Heroku. Это позволяет сделать простой, но мощный pipeline для этого проекта.

Мультиплейер

В Wikirace можно играть не только одному, но и в большой компании. Играйте со всеми Вашими друзьями. Кто дойдёт до цели первый, наберёт больше очков!
